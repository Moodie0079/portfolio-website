What CourseScout Does

CourseScout is a course insight platform that helps university students make informed decisions about their course selections. Currently focused on Carleton University with plans to expand to other universities, the platform analyzes years of student conversations on r/CarletonU to extract real student experiences and provide comprehensive course insights. The modular architecture enables easy adaptation to other universities' course catalogs and subreddit communities.

Challenges & Solutions

Initial Live API Approach

Problem: Initial version made live Reddit API calls for each course request, taking several minutes per course and hitting rate limits frequently, making the system slow and inefficient

Solution: Completely redesigned to use a database-driven architecture where data is pre-crawled and stored, reducing response times from minutes to 3-5 seconds

Rate Limiting During Database Crawling

Problem: Reddit API rate limits during the bulk crawling process for populating the database caused failures and slow data collection

Solution: Implemented intelligent rate limiting with header tracking, added retry logic with exponential backoff, and built progress tracking to resume interrupted crawling jobs

Professor Name Extraction Issues

Problem: AI extracting invalid professor names (single letters, generic terms like "professor")

Solution: Added comprehensive filtering to reject invalid names, required minimum name length and first+last name format, filtered out generic terms like "instructor", "teacher", "prof"

Bulk Processing Reliability

Problem: Bulk crawling jobs failing mid-process, losing progress

Solution: Implemented persistent progress tracking with JSON files, added resume capability for interrupted jobs, built comprehensive error reporting and retry logic
