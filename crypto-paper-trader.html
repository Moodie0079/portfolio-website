Crypto Paper Trader

A real-time cryptocurrency paper trading simulator for practicing trades with live market prices.

View on GitHub

Project Overview

This is a paper trading app built with Ruby on Rails and React where you can practice trading Bitcoin, Ethereum, and Solana with real market prices. It connects to Binance's live price feed and updates your portfolio value in real-time as prices change.

How It Works

The Rails backend handles all the trading logic and database operations. It connects to Binance's WebSocket stream to get live cryptocurrency prices, then broadcasts those prices to the React frontend through ActionCable (Rails' built-in WebSocket tool).

When you buy or sell crypto, the backend validates you have enough balance, updates your holdings in MySQL, and calculates your cost basis for profit/loss tracking. All trades happen inside database transactions so nothing breaks if something goes wrong mid-trade.

The React frontend shows your portfolio value, cash balance, and profit/loss for each position. Everything updates automatically as prices change - no refresh needed.

Details

Backend

Built with Ruby on Rails and MySQL. The backend has four API endpoints:

  • GET /portfolio - returns your current balance, holdings, and market prices
  • POST /trade - buy crypto
  • POST /sell - sell crypto
  • POST /reset - clear everything and start over

A background service connects to Binance's WebSocket stream and listens for price updates on BTCUSDT, ETHUSDT, and SOLUSDT. When prices change, it updates the database and broadcasts the new prices through ActionCable to all connected browsers.

Database transactions ensure buy/sell operations either complete fully or roll back entirely.

Frontend

Built with React and TypeScript. On load, it fetches your portfolio data then opens a WebSocket connection to receive live price updates.

As prices stream in, the UI recalculates your portfolio value and profit/loss for each position. Trade confirmations show up as toast notifications that fade out automatically.
