Student Grade Management System

A full‑stack web application that simplifies grade entry and database management for teachers and admins.

Project Overview

This app lets admins securely sign in and upload a single CSV to populate the database with student records, courses, grades, teacher, schedules, email addresses, etc. Teachers can then log in to adjust marks through a Bootstrap interface with all changes updating the database. This was developed using HTML, CSS, JavaScript, PHP, and MySQL.

Key Features

  • Developed a website for managing student grades, using HTML, JavaScript, PHP, CSS, and MySQL.
  • Designed a secure system for teachers and staff to input grades, preventing SQL injection.
  • Implemented login features to log in as either admin or teacher.
  • Admins have the ability to modify database entries and manage user permissions through a dashboard.
  • Admins could import CSV files to create a database of students, grades, teachers, classes, courses, emails etc.
  • Gained experience with full-stack web development and database management through this project.
